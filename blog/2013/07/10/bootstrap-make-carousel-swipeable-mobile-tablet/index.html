Bootstrap 2.3: Make the carousel swipeable on mobile & tablet

Posted 10 Jul 2013

Making the bootstrap carousel swipeable on mobile devices is a breeze. First grab a copy of the jQuery touchSwipe plugin.

Then use their slideLeft and slideRight callbacks to detect each gesture and respond to each by telling the carousel to go to the next/previous slide. Here’s the code:

1 $('#myCarousel').swipe( {
2     swipeLeft: function() {
3         $(this).carousel('next');
4     },
5     swipeRight: function() {
6         $(this).carousel('prev');
7     },
8     allowPageScroll: 'vertical'
9 });

Also set the allowPageScroll option to vertical as above to make sure that scrolling downwards isn’t blocked.

And that’s all there is to it. Happy coding!
