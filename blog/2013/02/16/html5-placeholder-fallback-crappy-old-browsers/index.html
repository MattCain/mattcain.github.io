HTML5 placeholder fallback for crappy old browsers

Posted 16 Feb 2013

One of the best features of HTML 5 is the input placeholder. All you have to do is:

<input type="text" placeholder="Enter some text!">

And you get a nice faded piece of text in your input that disappears to allow you to enter your text, like this:

You will have seen this used across the web, especially for search bars. The problem is that many citizens of the web are still using crappy, outdated browsers that don't support HTML5, and you may choose to implement a fallback for them so that they get the same experience.

Using the brilliant Modernizr library, you can easily detect a browser's missing features.

The below script will detect if the user's browser supports the placeholder attribute and will use javascript/jQuery to imitate it if it doesn't. Read the comments in the code to see what is going on.

 1 // If placeholder isn't supported.
 2 if (!Modernizr.input.placeholder) {
 3     // For every element that has a placeholder attribute
 4     $('[placeholder]').each(function() {
 5         var $this = $(this),
 6             placeholderValue = $this.attr('placeholder');
 7 
 8         // if field is empty, put the placeholder in it
 9         if ($this.val() == '') {
10             $this.val( placeholderValue );
11             $this.addClass('hasPlaceholderText');
12         }
13         // Add/remove placeholder on focus/blur
14         $this.focus(function() {
15             // Hide the placeholder so the user can enter
16             // their own text
17             if ($this.val() == placeholderValue) {
18                 $this.val('');
19                 $this.removeClass('hasPlaceholderText');
20             }
21         }).blur(function() {
22             // If the user didn't enter any text, show the
23             // placeholder text again.
24             if ($this.val() == '' || $this.val() == placeholderValue) {
25                 $this.val(placeholderValue);
26                 $this.addClass('hasPlaceholderText');
27             }
28         });
29 
30         // If the user submits the form, remove the placeholder
31         // if it is still there.
32         $this.closest('form').submit(function() {
33             if ($this.val() == $this.attr('placeholder')) {
34                 $this.val('');
35             }
36         });
37     });
38 }

Then all you have to do is add the following rule to your css to make your fake placeholder text gray:

.hasPlaceholderText {
    color: #aaa;
}

I hope this helps you out.
